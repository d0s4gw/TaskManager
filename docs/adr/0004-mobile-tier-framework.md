# ADR 0004: Mobile Tier Framework (Flutter)

## Status
Accepted

## Context
Require a high-performance mobile application for iOS and Android.

## Decision
Use **Flutter** for cross-platform development.

## Rationale
- **Performance**: Native-level performance from a single codebase.
- **UI Consistency**: Rich set of widgets for a premium look and feel.
- **Productivity**: Fast development cycles with hot reload.
