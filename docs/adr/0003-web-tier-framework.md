# ADR 0003: Web Tier Framework (Next.js)

## Status
Accepted

## Context
Need a modern, SEO-friendly, and high-performance web dashboard.

## Decision
Use **Next.js 16+** with the **App Router** and **Tailwind CSS**.

## Rationale
- **Performance**: Built-in optimization for routing and assets.
- **Security**: Seamless integration with Firebase Hosting and App Check.
- **Developer UX**: Modern React patterns and easy deployment.
